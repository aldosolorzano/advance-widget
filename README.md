# README

## Dependencies
  * Ruby 2.3.1
  * Rails 5.0.2
  * Ngrok

## Installation

Bundle ruby gems:
```
 $ bundle install
```  
Database creation:
```
 $ rails/rake db:create
```
How to run the test suite:
```
  $ rails s  
  $ ngrok http 3000
  
  Go to the documents_controller.rb and change the callback: ngrok link
```
